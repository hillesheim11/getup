## ðŸš€ Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/hillesheim11/getup.git && cd getup
```

Follow the steps below
```bash
# Install the dependencies
$ yarn

# Start the project
$ yarn dev
```
